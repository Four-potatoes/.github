# Progate Hackathon in Seoul

## Team name : 사차원 감자들🥔

**최웅식** : README.md로 프로젝트 전체 기획과 설계을 맡고 프로젝트를 총괄하며 json 데이터 제작 보조<br>
**제준혁** : JavaScript를 사용해서 프로젝트의 기능을 대부분 개발하고 초기 디자인 제작과 이미지 수집을 보조<br>
**윤태웅** : 팀원들이 만들어준 자료를 claude를 통해서 TypeScript로 통합하고 바이브코딩으로 채우기 어려운 부분은 직접 수정<br>
**최성우** : 프로젝트 전체 UI/UX 디자인을 관리하고 json 데이터 제작과 이미지를 수집
